# Welcome App - Basic Kotlin Android Application

This is a simple single-page Android application developed using Kotlin. The app allows the user to enter their name into an input field, and displays a personalized welcome message when the user clicks the **Submit** button. 

## Features:
- **EditText**: Allows the user to enter their name.
- **Submit Button**: Submits the entered name for validation.
- **TextView**: Displays a personalized welcome message once the name is entered correctly.

## Functionality:
1. The user enters their name in the **EditText** input field.
2. Upon clicking the **Submit** button:
   - If the input field is empty, the app prompts the user to enter their name.
   - If the input field is not empty, the app displays a personalized welcome message, e.g., “Welcome, [User’s Name]!”

## Concepts Demonstrated:
- **Button Click Event Handling**: Handling user interaction with a button.
- **Input Validation**: Using a simple `if` statement to check whether the user has entered a name.
- **Dynamic TextView Updates**: Updating the content of a `TextView` based on user input.
- **Working with User Input**: Retrieving and processing text entered by the user in Android.
