# Android-Studio-App-1
Basic Kotlin App 1 - Welcome App

This is a simple single-page Android application developed using Kotlin. The app allows the user to enter their name into an input field. When the user clicks the Submit button, the application checks whether the input field is empty using a basic if statement.
 - If the name field is empty, the app displays a message prompting the user to enter their name.
 - If the name is entered correctly, the app displays a personalized welcome message such as:
   “Welcome, [User’s Name]!”
The application features a minimal and clean user interface, consisting of:
 - An EditText for name input
 - A Submit button
 - A TextView to display the welcome message
This app demonstrates basic Kotlin programming concepts including:
 - Handling button click events
 - Using an if statement for input validation
 - Updating a TextView dynamically
 - Working with user input in Android
